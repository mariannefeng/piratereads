A public API for fetching Goodreads shelf data. For full interactive API reference, check out [piratereads Swagger](https://api.piratereads.com/swagger/index.html). Right now it supports:

- Fetching a user's currently reading books
- Fetching a user's read books
- Fetching a user's want to read books
