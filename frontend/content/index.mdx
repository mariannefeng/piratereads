This is a public API for fetching Goodreads shelf data. For the full interactive API reference, check out the [Swagger](https://api.piratereads.com/swagger/index.html). It supports:

- Fetching a user's currently reading books
- Fetching a user's read books
- Fetching a user's want to read books
