## Docker Image

Docker image is at [ghcr.io/mariannefeng/piratereads](https://github.com/mariannefeng/piratereads/pkgs/container/piratereads).

## Running the container

<Tabs items={["Docker", "Docker Compose"]}>
<Tab value="Docker">

Pull and run the image:

```bash
docker pull ghcr.io/mariannefeng/piratereads:latest

docker run -d \
  --name piratereads \
  --restart unless-stopped \
  -p 8080:8080 \
  ghcr.io/mariannefeng/piratereads:latest
```

</Tab>
<Tab value="Docker Compose">

Create a `docker-compose.yml`:

```yaml
services:
  piratereads:
    image: ghcr.io/mariannefeng/piratereads:latest
    container_name: piratereads
    restart: unless-stopped
    ports:
      - "8080:8080"
```

Then run:

```bash
docker compose up -d
```

</Tab>
</Tabs>

The API will be available at `http://localhost:8080`.
